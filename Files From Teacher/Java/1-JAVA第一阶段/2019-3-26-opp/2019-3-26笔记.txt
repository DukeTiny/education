抽象
abstract
声明类
声明方法

1、当一个类中有抽象方法，这个必须是抽象类。
2、抽象类不能实例化对象。必须被继承。
3、抽象类中可以没有抽象方法。
4、当一个类继承抽象类的时候，必须重写父类所有的抽象方法，如果没有重写，则子类必须声明成抽象类。
